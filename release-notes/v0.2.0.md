## CPA Android v0.2.0

本版本提供经过清理和自动验证的 Debug APK，主要更新：

- 统计页价格配置改为模型下拉框和一套共享输入区。
- 成本统计支持“全部模型”和单个模型筛选，并同步更新成本趋势和估算金额。
- CPA Usage Keeper 密码与 CLI Proxy API Management Key 分开保存。
- Management Key 使用隐藏输入，新增退出并清除本地凭据。
- 修复 Android Lint 问题并增加自动化测试、构建和 APK 发布流程。

验证结果：

- 10 个单元测试通过。
- Android Lint 无问题。
- Debug APK 构建成功。

> HTTP 仅建议用于可信局域网；公网或不可信网络请使用 HTTPS。
