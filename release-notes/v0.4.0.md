## CPA Android v0.4.0

本版本完成第一阶段产品化调整，重点改善首次配置、日常查看效率和慢网络体验：

- 首次打开改为两步引导：先验证 CPA Usage Keeper，再配置可选的 CLI Proxy API 账号服务。
- 下一步前必须通过真实 `/api/v1/status` 请求；空地址、示例地址、登录失败或连接失败都不会覆盖配置。
- 底部导航精简为 `总览 / 账号 / 用量 / 更多` 四栏。
- 总览新增服务、认证、请求可靠性和账号服务配置的健康结论，并提供账号、日志快捷操作。
- 用量页整合趋势成本、模型和凭证；更多页整合日志和设置。
- 刷新改为渐进加载：先展示会话、状态、模型和价格，再加载完整 `/api/v1/usage`。
- 新增轻量总览缓存，启动时明确标记缓存及原始范围；完整在线用量成功后才覆盖缓存。
- 完整用量请求失败时保留上次成功总览，不再用空统计覆盖。
- 保留现有加密凭据和登录状态恢复机制，正常重开无需重新连接和登录。
