## CPA Android v0.3.2

本版本改善重新打开应用时的连接和登录状态恢复：

- 登录状态和服务地址改为同步写入本机存储，减少应用很快退出时尚未落盘的情况。
- Android Keystore 临时解密失败时不再立即删除已保存凭据，后续请求会自动重试恢复。
- 网络请求发送前会在内存密码为空时重新读取加密凭据。
- 登录页明确说明凭据保存范围，以及卸载、清除数据和 APK 签名变化的影响。

验证结果：

- 13 个单元测试通过。
- Android Lint 无问题。
- Debug APK 构建成功。
- 与已发布 `v0.3.1` APK 的签名证书 SHA-256 一致，可直接覆盖安装并保留应用数据。
- APK SHA-256：`8C80EA6FE3E6E5C79A28172F95949FA2202E4D6C8552B20CB7357C7D95BDF838`（102933 字节）

> APK 更新必须保持签名一致才能保留应用数据。此项目当前提供 Debug 签名 APK，仅用于测试和自行安装。
