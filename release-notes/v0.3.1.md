## CPA Android v0.3.1

本版本提供经过自动验证的 Debug APK，主要更新：

- Android 端保存模型价格时，通过 `PUT /api/v1/pricing/{model}` 写回 CPA Usage Keeper。
- 服务端价格读取成功后，以服务端完整价格表为准，避免 Android 保留过期的本地价格。
- 价格写入失败时不修改本地价格，避免网页端与 Android 端产生无提示的分叉状态。
- 增加 Codex 认证文件启用和禁用控制，禁用后的账号仍可见并可重新启用。
- 修复 `used_percent = 1` 被错误解析为 100% 的额度显示问题。
- 额度刷新增加一次自动重试，并在临时失败时保留最后一次有效数据。

同步规则：网页端和 Android 端共用服务端价格表；另一端刷新后即可读取最新价格，冲突时以最后一次成功保存为准。
